# Invocation Examples

## 1. Rewrite a Normal Answer

```text
Use $classical-chinese-minimal-output to rewrite the following answer into austere Classical Chinese. Keep the meaning unchanged and do not add new facts.
```

## 2. Chinese Prompt

```text
请用最简雅正文言改写下文，勿增新义，勿赘述。
```

## 3. Technical Mixed Mode

```text
Use $classical-chinese-minimal-output and preserve all code, commands, file paths, and API names exactly as written.
```

## 4. Very Short Final Answer

```text
Use $classical-chinese-minimal-output. Answer in no more than three short sentences.
```

## 5. Summary Example

Input:

```text
请总结这次讨论：项目可以继续做，但要先补测试、确定接口、压缩范围。
```

Possible output:

```text
此议可续行，然先须补测试、定接口、敛范围。
```
